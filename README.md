# HistoryCloudBot
This is a full-stack React Native app with a 3D globe, Smithsonian API integration, and OpenAI GPT for historical queries.

### Features:
- **Interactive 3D Globe** using Three.js
- **Smithsonian Data Integration** via GitHub repositories
- **AI-powered responses** using OpenAI GPT

### Setup Instructions
1. Install dependencies:
```bash
yarn install
```
2. Run the backend:
```bash
yarn start
```
3. Launch the Expo frontend using Expo Go or any other mobile emulator.
